# Financial Performance Dashboard

## Overview

This project analyzes sales and profitability across products, countries, and customer segments using Power BI.

## Tools Used

* Power BI
* Excel
* DAX

## Key Metrics

* Total Sales: €118.73M
* Total Profit: €16.89M
* Profit Margin: 14.23%

## Key Insights

* Government segment contributes the largest share of sales.
* Paseo is the highest-profit product.
* October recorded the highest monthly sales.
* USA and Canada are the strongest markets.

## Dashboard Features

* KPI Cards
* Geographic Analysis
* Product Profit Analysis
* Profit Margin Analysis
* Interactive Filtering
